Detailed Description

template<typename ITYPE = int, typename DTYPE = double>
class knncolle::knn_base< ITYPE, DTYPE >

Virtual base class defining the knncolle interface.

Defines the minimum set up methods supported by all subclasses implementing specific methods.

Template Parameters
ITYPEInteger type for the indices.
DTYPEFloating point type for the distances.

Find the nearest neighbors of a new observation.

Parameters
queryPointer to an array of length equal to ndims(), containing the coordinates of the query point.
kThe number of neighbors to identify.
[out]indicesPointer to a vector to store the index of the nearest neighbors. Vector is resized to length no greater than k (but possibly less, if the total number of observations is less than k). If NULL, no indices are returned.
[out]distancesPointer to a vector to store the distances to the nearest neighbors. Each distance corresponds to a neighbor in indices. Guaranteed to be sorted in increasing order. If NULL, no distances are returned.
Returns
If report_indices = true, *indices is filled with the identities of the k nearest neighbors. If report_distances = true, *distances is filled with the distances to the k nearest neighbors.
