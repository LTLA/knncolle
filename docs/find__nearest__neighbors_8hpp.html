NeighborList< INDEX_t, DISTANCE_t > knncolle::find_nearest_neighbors ( const Base< InputINDEX_t, InputDISTANCE_t, InputQUERY_t > *  ptr,
int  k 
)

Find the nearest neighbors within a pre-built index. This is a convenient wrapper around Base::find_nearest_neighbors that saves the caller the trouble of writing a loop.

Template Parameters
INDEX_tInteger type for the indices in the output object.
DISTANCE_tFloating point type for the distances in the output object
InputINDEX_tInteger type for the indices in the input index.
InputDISTANCE_tFloating point type for the distances in the input index.
QUERY_tFloating point type for the query data in the input index.
Parameters
ptrPointer to a Base index.
kNumber of nearest neighbors.
Returns
A NeighborList of length equal to the number of observations in ptr->nobs(). Each entry contains the k nearest neighbors for each observation, sorted by increasing distance.
